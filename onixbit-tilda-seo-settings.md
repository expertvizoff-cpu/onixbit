# SEO-настройки страниц Tilda

Дата: 2026-06-16

Использовать в настройках каждой страницы Tilda: title, description и OG image.

Общий OG image:

- `https://static.tildacdn.pub/tild6230-3532-4239-a233-316163323733/onixbit-og-preview-1.png`

Этот OG image уже используется на опубликованной главной странице Tilda. Его можно поставить на все внутренние страницы, чтобы закрыть пустые OG-картинки.
